## Purpose

Track all tasks related to this project, with completion status and dates.

## Structure

Each task is a checkbox with an optional completion date. Tasks are grouped by category.

## Formatting Rules

- Use `[x]` for completed tasks, `[ ]` for pending tasks.
- Append ` YYYY-MM-DD` to completed tasks to record when they were done.
- Group tasks under category headings.
- Prefix category groups with `[CAT]` in comments for easy filtering.
